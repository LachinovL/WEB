1. Skeleton, bootstrap, gumby. Фреймворки значительно сокращают работу при верстке страницы. Минусы: сложно разобраться в чужом коде. 
2. React, angular. Упрощает работу при написании кода. 
3. Принципы адаптивной верстки: относительные единицы измерения, максимальные и минимальные значения, вложенные объекты,  точки разрыва
4. Принципы БЭМ: разделение на независимые блоки, любой блок кода может быть использован повторно, CSS-свойства для блоков, элементов и модификаторов описываются только через классы
Другие методологии верстки: MCSS, OOCSS, Atomic CSS
